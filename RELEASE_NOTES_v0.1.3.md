# EcrisTonHistoire — v0.1.3

## Mise à jour automatique

Cette version introduit le **système de mise à jour automatique** de l'application.

À partir de maintenant, plus besoin de venir vérifier manuellement si une nouvelle version est disponible.
Au démarrage, l'appli vérifie silencieusement GitHub. Si une mise à jour existe, elle se télécharge en arrière-plan pendant que tu travailles, puis s'installe à la fermeture — ou immédiatement si tu le souhaites.

### Comment ça se passe

1. Tu lances EcrisTonHistoire normalement
2. Si une mise à jour est disponible, une bannière apparaît en haut de l'écran
3. Le téléchargement se fait tout seul, avec une barre de progression
4. Une fois prêt, tu peux choisir d'**installer maintenant** (l'appli redémarre) ou laisser l'installation se faire **à la prochaine fermeture**

### Notes d'installation

> ⚠️ Ce changement nécessite de **désinstaller l'ancienne version** avant d'installer celle-ci.
> Le format du programme d'installation est passé de `.msi` à `.exe` — les deux ne peuvent pas coexister.
> **Tes données sont conservées**, seul le programme change.

Télécharge le fichier `EcrisTonHistoire Setup 0.1.3.exe` ci-dessous et lance l'installation.
